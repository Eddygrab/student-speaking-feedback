# 학생 영어 말하기 피드백 앱

학생이 말한 영어 단어/문장을 브라우저 음성인식으로 받아 적고, 목표 문장과 비교해서 간단한 피드백을 보여주는 React + Vite 앱입니다.

## 설치
```bash
npm install
```

## 실행
```bash
npm run dev
```

## 배포
GitHub에 올린 뒤 Vercel에서 Import Project로 연결하면 됩니다.

## Google Sheets 기록 저장
Apps Script 웹앱 URL이 이미 `src/App.jsx`에 반영되어 있습니다.
기록 저장 버튼을 누르면 Google Sheets에 저장되도록 구성되어 있습니다.
